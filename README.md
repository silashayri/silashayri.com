# Silasha | Software Engineer Portfolio

Siyah arka planlı, matrix efektli, minimal ve etkileyici tek sayfa kişisel web sitesi.

## Özellikler

- Matrix rain animasyonu arka planda
- Glitch efektli isim animasyonu
- Typing animasyonu ile dinamik metinler
- Hover efektli sosyal medya linkleri
- Tam responsive tasarım
- Siyah arka plan, yeşil neon tema
- Sade ama etkileyici görünüm

## Özelleştirme

1. **index.html** dosyasında:
   - `SILASHA` kısmını kendi isminizle değiştirin
   - Link URL'lerini güncelleyin (GitHub, LinkedIn, Twitter, Email)
   - `yourusername` ve `your.email@example.com` kısımlarını değiştirin

2. **script.js** dosyasında:
   - `phrases` dizisindeki metinleri kendi ifadelerinizle değiştirin

3. **style.css** dosyasında:
   - Renkleri değiştirmek için `#0f0` (yeşil) değerini başka renk kodlarıyla değiştirebilirsiniz
   - Font boyutlarını ayarlayabilirsiniz

## Çalıştırma

```bash
# Basit HTTP server ile
python3 -m http.server 8000

# Veya
npx serve
```

Tarayıcınızda `http://localhost:8000` adresini açın.

## Teknolojiler

- HTML5
- CSS3 (Animasyonlar, Glitch efekti)
- Vanilla JavaScript (Matrix rain, Typing animasyonu)
- Canvas API
